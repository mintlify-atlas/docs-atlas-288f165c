## Enhanced Battle UI

Interfaz de batalla mejorada con:

- **Indicadores de efectividad**: Muestra super efectivo/poco efectivo
- **Backgrounds por tipo**: Fondos que reflejan el tipo del movimiento
- **Prompts visuales**: Botones de ayuda configurables
- **Stats visibles**: Modificadores de stats durante batalla

## Configuración

```ruby
# Settings module
Settings::USE_MOVE_TYPE_BACKGROUNDS = true
Settings::SHOW_TYPE_EFFECTIVENESS_FOR_NEW_SPECIES = false
Settings::UI_PROMPT_DISPLAY = 2  # 0=off, 1=always, 2=hide after 2s
```

## Movimientos Gen 9
