## Gen 8 Move Animation Project

**Actualizado 2022-04-20**: Más de 800 movimientos animados.

## Características

- Animaciones para todos los movimientos hasta Gen 8
- Efectos visuales mejorados
- Optimizadas para rendimiento
- Validaciones anti-crash

## Validaciones

**Mejorado en v1.1.0 y v1.0.8**:

- Validaciones para evitar errores
- Compatibilidad con bases sin animaciones
- Corrección de Triple Axel DBK (v1.1.2)

## Ubicación

```
Data/PkmnAnimations.rxdata
```

## Editor de animaciones

Incluye `animmaker.exe` para crear/editar animaciones:

```
animmaker.exe
animmaker.txt  # Instrucciones
```

## Créditos

Basado en Gen 8 Move Animation Project de la comunidad.

<Note>
Las animaciones se cargan automáticamente. No requiere configuración adicional.
</Note>
