## Map Exporter

**Incluido en v1.0**: Utilidad para exportar mapas.

## Mejoras v1.0.1

- Se guardan en carpeta separada
- No sobreescribe archivos existentes
- Nombres descriptivos automáticos

## Uso

```ruby
# Desde debug menu o script
pbExportMap(map_id)

# Exportar todos los mapas
pbExportAllMaps
```

## Ubicación

Mapas exportados en:
```
ExportedMaps/
└── Map001_PuebloInicio.png
└── Map002_Ruta1.png
```

## Usos

- Documentación
- Planificación
- Visualización completa
- Compartir diseños
