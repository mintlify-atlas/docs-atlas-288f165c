## Qué es el Pokévial

Sistema de curación portátil con cargas limitadas.

## Uso

1. Obtener el Pokévial (objeto clave)
2. Usar desde el menú
3. Seleccionar Pokémon a curar
4. Se consume una carga

## Recargar

Se recarga automáticamente al:
- Visitar un Centro Pokémon
- Usar estaciones de recarga

## Pokévial infinito

**Agregado en v1.1.2**: Opción para cargas infinitas.

```ruby
Settings::INFINITE_POKEVIAL = true
```
