## ¿Qué es La Base de Sky?

La Base de Sky es una versión mejorada y extendida de **Pokémon Essentials v21.1** completamente en español, creada por [Skyflyer](https://twitter.com/Sky_fangames) y [DPertierra](https://github.com/dpertierra). Es la base más completa disponible para desarrolladores que quieren crear fangames de Pokémon con todas las características modernas.

<Note>
Pokémon Essentials es un kit de desarrollo de juegos construido sobre RPG Maker XP que permite crear fangames de Pokémon con todas las mecánicas oficiales.
</Note>

## Características principales

### Pokédex Avanzada
Búsqueda instantánea (tecla D), filtros por tipo, generación y estado de captura. Visualización completa de movimientos, habilidades y evoluciones.

### Sistema de Combate Mejorado
- Enhanced Battle UI con indicadores visuales
- Efectividad de tipos mostrada durante la batalla
- Animaciones de movimientos actualizadas (2022-04-20)
- Soporte completo para mecánicas de Gen 9

### Gestión de Pokémon
- **PC mejorado**: Ordenar cajas, buscar Pokémon (tecla W), marcar favoritos
- **Incubadora de huevos**: Sistema integrado que permite seleccionar huevos desde el PC
- **Visualización de IVs/EVs**: Ver estadísticas individuales y de esfuerzo
- **Intercambio desde PC**: Intercambiar Pokémon sin sacarlos del almacenamiento

### Regalo Misterioso Renovado
Sistema completamente rehecho que permite crear regalos por internet con contraseña. Soporta Pokémon, items, y huevos con configuración avanzada.

### Herramientas para Desarrolladores

<Accordion title="Debug Passability">
  Visualiza la pasabilidad de tiles para encontrar errores de mapeo. Actívalo con F3 para abrir el terminal de debug.
</Accordion>

<Accordion title="Map Exporter">
  Exporta mapas completos a imágenes PNG para documentación o planificación.
</Accordion>

<Accordion title="Turbo Speed">
  Sistema de velocidad configurable con 3 niveles. Opción para activar turbo solo en combates.
</Accordion>

## ¿Quién debería usar esta base?

### Desarrolladores principiantes
La base incluye mapas de ejemplo, NPCs demostrativos, y documentación completa en español para aprender rápidamente.

### Desarrolladores experimentados
Arquitectura modular con plugins, Ruby Standard Library 3.3.0, y herramientas avanzadas de debugging.

### Equipos de desarrollo
Sistema de contribución con Git, documentación de APIs, y estructura organizada por plugins.
