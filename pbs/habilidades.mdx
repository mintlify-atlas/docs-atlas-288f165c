## Archivo abilities.txt

```
[OVERGROW]
Name = Espesura
Description = Potencia movimientos Planta en apuros.
```

### Estructura simple

Cada habilidad tiene:
- ID interno
- Nombre visible
- Descripción

La lógica se implementa en scripts.

## Gen 9

Todas las habilidades de Paldea incluidas.

## Implementación

Las habilidades se implementan en:

`Data/Scripts/029_Battler/Battler_Abilities.rb`

```ruby
BattleHandlers::AbilityOnSwitchIn.add(:INTIMIDATE,
  proc { |ability, battler, battle|
    battle.pbShowAbilitySplash(battler)
    # Lógica de la habilidad
  }
)
```
