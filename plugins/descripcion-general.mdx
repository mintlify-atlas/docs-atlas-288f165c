## Introducción

La Base de Sky v21.1 incluye un sistema completo de plugins que extienden y mejoran las funcionalidades base de Pokémon Essentials. Estos plugins han sido cuidadosamente seleccionados e integrados para proporcionar una experiencia de desarrollo mejorada.

## Plugins Incluidos

### Plugins MUI (Modular UI)

Estos plugins requieren el sistema **Modular UI Scenes** como dependencia base:

- **[MUI] Enhanced Pokemon UI** (v1.0.6) - Interfaz mejorada para Pokémon
- **[MUI] Pokedex Data Page** (v2.0.4) - Página de datos detallados para la Pokédex
- **Lin's IV EV Summary Screen** (v1.2.1) - Pantalla de resumen de IVs y EVs

### Plugins de Utilidades

- **Luka's Scripting Utilities** (v4.1.0) - Utilidades de scripting avanzadas
- **Marin's Scripting Utilities** - Utilidades complementarias de scripting
- **Tileset Rearranger** - Herramienta para reorganizar tilesets

### Plugins de Funcionalidad

- **Box Auto-Sort** (v1.0.0) - Sistema de ordenamiento de cajas
- **Easy Mouse** - Soporte mejorado para mouse
- **Event Reporting** - Sistema de reporte de eventos para debugging

## Gestión de Plugins

### Habilitar/Deshabilitar Plugins

Puedes controlar qué plugins están activos editando el campo `Disabled` en el archivo `meta.txt` de cada plugin:

```ruby
# En Plugins/[Nombre del Plugin]/meta.txt
Name         = Nombre del Plugin
Version      = 1.0.0
Disabled     = true  # true para deshabilitar, false o vacío para habilitar
```

### Prioridad de Carga

Algunos plugins definen un orden de carga específico:

```ruby
First        = True   # Se carga primero
Priority     = 3      # Prioridad de carga (números mayores = mayor prioridad)
```

<Warning>
No deshabilites plugins que sean dependencias de otros plugins activos. Esto puede causar errores en el juego.
</Warning>

## Configuración Global

La mayoría de los plugins pueden configurarse a través del módulo `Settings` en:

```
Plugins/[Nombre del Plugin]/[000] Settings.rb
```

O en el archivo global:

```
Data/PluginScripts/PluginSettings.rb
```

## Verificación de Plugins

El sistema incluye verificación automática de compatibilidad de versiones:

```ruby
# Verifica si un plugin está instalado
PluginManager.installed?("Nombre del Plugin")

# Verifica versión mínima
PluginManager.installed?("Nombre del Plugin", "1.0.0", true)

# Obtiene la versión instalada
PluginManager.version("Nombre del Plugin")

# Obtiene el enlace del plugin
PluginManager.link("Nombre del Plugin")
```
