## BattleSettings

Archivo: `003_BattleSettings.rb`

## Constantes

### Animaciones

```ruby
# Duración de animaciones
BATTLE_ANIMATION_SPEED = 1.0

# Mostrar animaciones
SHOW_MOVE_ANIMATIONS = true
```

### Experiencia

```ruby
# Escala de experiencia
EXP_SCALE = 1.0

# Máximo nivel
MAXIMUM_LEVEL = 100
```

### Mecánicas

```ruby
# Mega Evolución
ENABLE_MEGA_EVOLUTION = true

# Dinamax (si está implementado)
ENABLE_DYNAMAX = false
```
