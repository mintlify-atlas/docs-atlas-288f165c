## Clases core

Archivo: `011_Game classes/`

### Game_Player

Maneja el jugador en el mapa:

```ruby
$game_player.moveto(x, y)
$game_player.turn_up
$game_player.jump(x_plus, y_plus)
```

### Game_Map

Maneja el mapa actual:

```ruby
$game_map.map_id
$game_map.name
$game_map.metadata
```

### Game_System

Sistema global:

```ruby
$game_system.bgm_play(bgm)
$game_system.bgs_play(bgs)
```
